개인정보 보호정책

최종 업데이트: 2025년 7월 31일

요약: 이 개인정보 보호정책은 본 웹사이트가 어떻게 귀하의 개인정보를 수집, 사용, 보호하는지 설명합니다.

1. 수집하는 개인정보

본 서비스는 다음과 같은 개인정보를 수집할 수 있습니다:

  • 자동 수집 정보: IP 주소, 브라우저 유형, 방문 시간, 페이지 조회 기록
  • 선택적 제공 정보: 이메일 주소, 이름 (문의 시에만)
  • 쿠키 및 유사 기술: 웹사이트 기능 향상을 위한 기술적 쿠키

2. 개인정보 사용 목적

수집된 개인정보는 다음 목적으로만 사용됩니다:

  • 서비스 제공 및 웹사이트 운영
  • 사용자 문의 응답 및 고객 지원
  • 웹사이트 개선 및 분석
  • 법적 의무 이행

3. 개인정보 보관 및 삭제

  • 보관 기간: 수집 목적 달성 시까지 또는 법적 요구사항에 따라
  • 자동 삭제: 로그 데이터는 30일 후 자동 삭제
  • 요청 시 삭제: 사용자 요청 시 지체 없이 삭제

4. 개인정보 공유 및 제3자 제공

본 서비스는 다음의 경우를 제외하고는 개인정보를 제3자에게 제공하지 않습니다:

  • 사용자의 명시적 동의가 있는 경우
  • 법적 요구 또는 정부 기관의 요청이 있는 경우
  • 서비스 제공을 위해 필요한 업체 (GitHub Pages, 분석 도구 등)

5. GitHub Pages 관련 정보

본 웹사이트는 GitHub Pages에서 호스팅됩니다:

  • GitHub의 개인정보 보호정책이 적용됩니다
  • GitHub는 기술적 로그를 수집할 수 있습니다
  • 자세한 내용은 GitHub 개인정보 보호정책을 참조하세요

6. 쿠키 정책

  • 필수 쿠키: 웹사이트 기본 기능을 위해 사용
  • 분석 쿠키: 웹사이트 사용 패턴 분석 (선택사항)
  • 쿠키 관리: 브라우저 설정에서 쿠키를 비활성화할 수 있습니다

7. 사용자 권리

귀하는 다음과 같은 권리를 가집니다:

  • 열람권: 개인정보 처리 현황을 알 권리
  • 정정·삭제권: 개인정보의 수정이나 삭제를 요구할 권리
  • 처리정지권: 개인정보 처리 중단을 요구할 권리
  • 손해배상청구권: 개인정보 침해로 인한 피해 구제를 요구할 권리

8. 보안

개인정보 보호를 위해 다음과 같은 보안 조치를 취합니다:

  • HTTPS 암호화 통신
  • 접근 권한 제한
  • 정기적인 보안 점검
  • 개인정보 최소 수집 원칙

9. 정책 변경

본 개인정보 보호정책은 관련 법령이나 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경사항이 있을 경우 웹사이트를 통해 공지하겠습니다.
